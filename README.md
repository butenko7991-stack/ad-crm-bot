# AD-CRM-Bot

CRM-бот для управления продажами рекламы. Работает одновременно в **Telegram** и **Max** (ex-ICQ New, мессенджер VK). Полноценная CRM-система для команды менеджеров с автоматизацией, геймификацией и AI-тренером.

## 🛠 Технологии

| Компонент | Технология |
|-----------|-----------|
| Telegram-бот | [Aiogram 3.4+](https://docs.aiogram.dev/) |
| Max-бот | [maxapi 0.9+](https://github.com/max-messenger/max-botapi-python) |
| База данных | PostgreSQL + SQLAlchemy 2.0 (async) |
| Драйвер БД | asyncpg |
| HTTP | aiohttp |
| Конфиг | python-dotenv |
| Планировщик | APScheduler |
| AI-тренер | Claude API (Anthropic) |
| Аналитика каналов | Telemetr.io API |

## 🚀 Запуск

### 1. Переменные окружения

Создайте `.env` файл:

```env
BOT_TOKEN=ваш_токен_telegram_бота
MAX_BOT_TOKEN=ваш_токен_max_бота   # опционально, для сети Max
DATABASE_URL=postgresql+asyncpg://user:pass@localhost/ad_crm
ADMIN_IDS=123456789,987654321
ADMIN_PASSWORD=ваш_пароль
CLAUDE_API_KEY=ваш_ключ_claude   # опционально
TELEMETR_API_TOKEN=ваш_токен     # опционально
AUTOPOST_ENABLED=true
```

> **Токен для Max:** Зарегистрируйте бота через [@MaxBotAPI](https://max.ru/botapi) и добавьте полученный токен в `MAX_BOT_TOKEN`. Если переменная не задана, бот запустится только в Telegram.

### 2. Установка зависимостей

```bash
pip install -r requirements.txt
```

### 3. Запуск

```bash
python main.py
```

## ✅ Что реализовано

### 👑 Панель администратора (`handlers/admin.py`)

- **Управление каналами**
  - Добавление каналов (пересылка сообщения или @username)
  - Редактирование цен (1/24, 1/48, 2/48, нативная)
  - Авторасчёт цен по CPM и охвату
  - Обновление статистики подписчиков
  - Включение/отключение канала
  - Удаление канала

- **Оплаты**
  - Список заказов с загруженными скриншотами оплаты
  - Просмотр деталей заказа (клиент, канал, формат, сумма)
  - Подтверждение или отклонение оплаты с уведомлением клиента

- **Менеджеры**
  - Список менеджеров с уровнями и статистикой
  - Просмотр профиля менеджера
  - Повышение/понижение уровня (1–5)
  - Активация/деактивация аккаунта

- **Модерация постов**
  - Список запланированных постов на модерации
  - Просмотр содержимого поста
  - Одобрение (переводит в очередь) или отклонение

- **Соревнования**
  - Просмотр активных соревнований
  - Создание соревнования (название, даты, призовой фонд, метрика)

- **Статистика**
  - Общее число заказов, выручка, менеджеры, каналы

- **CPM тематик**
  - Просмотр CPM по всем 40+ тематикам

- **Настройки**
  - Статус автопостинга, Claude API, Telemetr API

### 💼 Кабинет менеджера (`handlers/manager.py`)

- Профиль (уровень, XP, баланс, продажи)
- Мои продажи (история заказов)
- Мои клиенты
- Шаблоны сообщений (3 встроенных)
- AI-помощник (Claude API)
- Запрос выплаты (сумма, способ, реквизиты)
- История выплат
- Таблица лидеров (по продажам / выручке / XP)
- Реферальная ссылка

### 🛒 Бронирование рекламы (`handlers/client.py`)

Полный FSM-флоу из 8 шагов:

1. Каталог каналов с ценами
2. Выбор канала (детали, аналитика)
3. Выбор даты из доступных слотов
4. Выбор времени
5. Выбор формата размещения
6. Ввод рекламного текста
7. Загрузка медиа (фото/документ)
8. Подтверждение + загрузка скриншота оплаты

### 📚 Обучение (`handlers/training.py`)

- 2 встроенных урока с квизами
- AI-тренер: диалог с Claude по темам продаж
- Прогресс обучения
- Обратная связь по ответам AI (👍/👎)

### 🎮 Геймификация (`services/gamification.py`)

| Уровень | Название | Комиссия | Мин. продаж |
|---------|----------|----------|-------------|
| 1 | 🌱 Стажёр | 10% | 0 |
| 2 | 🌿 Джуниор | 12% | 5 |
| 3 | 🌳 Мидл | 15% | 15 |
| 4 | 🏆 Сеньор | 18% | 30 |
| 5 | 👑 Лид | 25% | 50 |

- Начисление XP за продажи (10 базовых + 1 за каждые 100₽)
- Авто-повышение уровня при наборе XP
- Система достижений
- Таблица лидеров (продажи / выручка / XP)
- Создание ежемесячных соревнований

### 📊 Аналитика каналов (`services/telemetr.py`)

- Получение статистики через Telemetr.io API
- Подписчики, охват, ERR%, CI-индекс
- Поиск канала по username

### 🤖 AI-тренер (`services/ai_trainer.py`)

- Интеграция с Claude (claude-3-haiku)
- Системный промпт для обучения продажам рекламы
- Память разговора (последние 10 сообщений)
- Сохранение Q&A пар в БД для анализа
- Извлечение топиков из ответов

## 🗄 Схема базы данных

| Таблица | Описание |
|---------|----------|
| `channels` | Каналы для рекламы (цены, аналитика, категория) |
| `category_cpm` | CPM по тематикам (редактируется в боте) |
| `slots` | Временные слоты для размещения |
| `clients` | Рекламодатели |
| `managers` | Менеджеры (уровень, XP, баланс) |
| `orders` | Заказы на размещение |
| `manager_payouts` | Выплаты менеджерам |
| `scheduled_posts` | Очередь автопостинга |
| `competitions` | Соревнования менеджеров |
| `ai_insights` | Обратная связь по AI-ответам |
